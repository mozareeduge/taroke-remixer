# 11 — Research Rationale

## Media archaeology

Translate operations:

- Cue channel → procedural audition.
- Session/Arrangement → Pattern Matrix / Flow Score.
- Channel strip → stable lanes/inspectors.
- Stem separation → Unmix Line.
- Non-destructive editing → undo/Takes.
- Direct manipulation → immediate consequence/reorder.
- Modeless performance → edit/Cue while Surface continues.
- Waveform legibility → procedural Monitor.

Reject:

- fake turntables;
- decorative crossfaders;
- hidden-value knobs;
- BPM language;
- banks as stems;
- grammar as EQ;
- low-contrast neumorphism.

## Prior evidence

Earlier work established:

- shared core/artifact parity is essential;
- provenance matters;
- local-first export is a truth condition;
- high-level grouping can beat eleven equal tabs;
- shell design without real browser acceptance is insufficient;
- whole-app template rerendering constrains stable authoring.

## Claude implementation
