## Friction record

- completion;
- time;
- navigation count;
- unexpected scroll;
- hidden consequence;
- terminology hesitation;
- errors/recovery;
- undo;
- mobile reachability.

Hard failures:

- import appears replaced;
- consequence invisible;
- focus/caret lost;
- destructive action unrecoverable;
- drag-only;
- mobile control covered;
- weight/chance confused;
- artifact differs semantically.
